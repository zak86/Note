Наступні методи дозволяють встановлювати компоненти дати і часу :

setFullYear (рік [ , місяць, дата ] )
setMonth (місяць [ , дата ] )
SETDATE (дата)
setHours (година [ , хв , сек , мс ] )
setMinutes ( хв [ , с, мс ] )
setSeconds (сек [ , мс ] )
setMilliseconds ( мс)
SetTime ( в мілісекундах ) ( встановлює всю дату по мілісекунд з 01.01.1970 UTC )
Всі вони, крім SetTime ( ), мають також UTC - варіантом , наприклад : setUTCHours ( ) .
