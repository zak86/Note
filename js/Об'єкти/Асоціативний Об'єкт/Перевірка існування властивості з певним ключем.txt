Наприклад, є об'єкт person, і потрібно перевірити, чи існує в ньому властивість age.

Для перевірки існування є оператор in. Його синтаксис: "свойство" in obj, причому ім'я властивості - "береться в кавички", наприклад:

var person = { };

if ("age" in person) {
  alert("Свойство age существует!");
}

Втім, частіше використовується інший спосіб - порівняння значення з undefined.

Справа в тому, що в JavaScript можна звернутися до будь-якого властивості об'єкта, навіть якщо його немає. Помилки не буде.

Але якщо властивість не існує, то повернеться спеціальне значення undefined:

var person = { name: "Василий" };

alert(person.age === undefined); // true, свойства нет
alert(person.name === undefined); // false, свойство есть.
